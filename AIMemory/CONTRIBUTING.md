# Contributing to AIMemory

Thank you for your interest in contributing to AIMemory! We welcome contributions from the community to help make this project better.

## How to Contribute

1.  **Fork the repository**: Click the "Fork" button on the top right of the repository page.
2.  **Clone your fork**: `git clone https://github.com/YOUR_USERNAME/AIMemory.git`
3.  **Create a branch**: `git checkout -b feature/my-new-feature`
4.  **Make changes**: Write code, add tests, and update documentation.
5.  **Run Tests**: Ensure all tests pass by running `pytest`.
6.  **Commit changes**: `git commit -m "Add my new feature"`
7.  **Push to branch**: `git push origin feature/my-new-feature`
8.  **Submit a Pull Request**: Open a PR against the `main` branch.

## Development Setup

1.  Install dependencies: `poetry install`
2.  Run tests: `pytest`
3.  Run linter: `flake8` (if configured)

## Code Style

- Follow PEP 8 guidelines.
- Use meaningful variable and function names.
- Add docstrings to functions and classes.

## Reporting Issues

If you find a bug or have a feature request, please open an issue on the GitHub repository. Provide as much detail as possible, including steps to reproduce the issue.
